Unity Tools Built for Production Projects

These tools exist to reduce release risk, enforce clean workflows, and make build problems visible before they become expensive.

Build Size Guard

Detect unexpected build size increases before they reach players.

  • Snapshot size tracking
  • Diff-based comparison
  • CI-friendly reporting
Product Page Docs License

Shader Variant Budget & CI Guard

Track shader variant counts over time, compare builds, and stop regressions with budgets and CI gates.

  • Editor.log snapshot capture
  • Baseline versus current diffing
  • Budget enforcement in CI
Product Page Docs License

Unity Serialization Migration Guard

Protect serialized project data during refactors with audited mapping checks, scoped coverage scans, and cleanup readiness reporting.

  • Audit mappings for risk level and resolution issues
  • Scan scenes, prefabs, and ScriptableObjects for legacy tokens
  • Run batchmode audit, coverage, and cleanup checks
Product Page Docs License

Import Settings Validator & Fix

Enforce consistent import settings and fix drift across your Unity project.

  • Rule-based validation
  • Bulk auto-fix
  • Project-wide enforcement
Product Page Docs License

Third-Party Notices & Credits

Generate deterministic third-party notices and compliance manifests for Unity releases.

  • Automatic license aggregation
  • Deterministic output
  • Release-ready manifests
Product Page Docs License
