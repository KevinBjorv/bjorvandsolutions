Why teams use it

Release time is the worst moment to discover you do not know your attribution obligations.

Unity projects collect packages, plugins, embedded dependencies, native libraries, and copied utilities over time.

  • UPM packages.
  • Asset Store plugins.
  • Embedded dependencies.
  • Native libraries.
  • Copied utilities.

Most teams end up guessing which notices to include. This tool exists to replace that guesswork with evidence found in the project itself.

Evidence-based detection

Scan for the files and markers that actually support an attribution decision.

  • LICENSE files.
  • NOTICE files.
  • COPYING files.
  • Package manifests.
  • Known license markers.

If proof exists, the scanner reports it. If not, the item stays marked UNKNOWN.

Scan targets

Cover the dependency locations teams typically need to review.

  • UPM packages.
  • Asset Store imports.
  • Plugins folders.
  • Native binaries.
  • Embedded third-party folders.

Manual overrides

Handle edge cases when project evidence is incomplete.

  • License name.
  • Attribution.
  • Source URL.
  • Notes.

This keeps the workflow practical when a dependency needs cleanup before export.

Export output

Generate THIRD_PARTY_NOTICES and CREDITS faster.

  • Markdown output.
  • Text output.
  • Structured formats for pipeline usage.

That gives teams a practical starting point for review and release packaging.

Who it is for

Best fit for teams shipping commercial builds with dependencies.

  • Unity studios shipping commercial games.
  • Publishers requiring compliance files.
  • Teams preparing console submission.
  • Teams that do not want legal surprises late in release.

Who it is not for

Not aimed at projects with no dependency footprint.

  • Hobby projects with zero dependencies.
  • Teams willing to guess licenses.
  • Legal automation beyond evidence found in the project.

FAQ

Common questions before using it in a release workflow.

Is this legal advice?

No. It provides evidence-based detection from your project files.

Does it scan external websites?

No. It only scans your project.

Can I edit the output?

Yes. The output is editable before export.
