Unity Editor Tool

Unity Serialization Migration Guard

A safe workflow for serialized field migration and cleanup.

Use it to validate renamed fields, map risks to concrete actions, scan assets for legacy tokens, and confirm cleanup safety before removing migration paths.

Built from manual mappings, diff import, scope-based coverage, and batch checks for deterministic migration quality.

  • Manual + diff-based mapping import
  • Coverage-aware rescan and backup path control
  • Batchmode audit, coverage, and cleanup checks

What it is for

Keep serialized migration paths verifiable and reversible.

During large refactors, serialized field and type changes can persist in project assets long after code compiles. This tool reduces hidden breakage by surfacing mapping risks, confirming scope coverage, and proving when cleanup is safe.

  • Manual mapping and Git-diff import support for refactor workflows.
  • Asset scanning for scenes, prefabs, and ScriptableObjects before cleanup actions.
  • Re-serialize preview and backup-aware execution.
  • Batch export and deterministic exit codes for CI.

Workflow model

Use the same tabs you already use: Audit, Coverage, Reserialize, Cleanup, Settings.

  • Audit: validate field mappings and risk scores.
  • Coverage: scan legacy tokens across supported asset kinds.
  • Reserialize: build a scoped preview and run controlled updates.
  • Cleanup: evaluate whether mappings are safe to remove.
  • Settings: control defaults and persistence behavior.

Who it is for

Teams shipping refactors with serialized state.

  • Unity teams replacing fields, types, or namespaces at project scale.
  • Teams using CI where migration safety needs deterministic checks.
  • Projects that need explicit cleanup proofs before removing legacy compatibility.

Who it is not for

Not the first choice for non-editor, non-serialized migration work.

  • Gameplay runtime tuning workflows with no serialized refactor risk.
  • Projects not using Force Text serialization.
  • One-off projects that do not remove legacy serialized mappings.
