Tools Directory

Unity tools with direct paths to docs, licenses, and support.

Use this page as the primary index for all current Bjorvand Solutions tools. Every tool card includes product context and direct links to the pages needed during evaluation, setup, and support.

Current Tool Pages

Each entry includes what the tool is for, who it is for, and direct links to product, docs, license, storefront, or support pages.

Build Size Guard

Detect and prevent build size regressions with snapshot capture, diffing, budgets, and CI report outputs.

Who it is for: Unity teams that need deterministic size guardrails before release.

Product Docs License Store

Shader Variant Budget & CI Guard

Track shader variant counts over time, compare builds, enforce budgets, and stop shader variant regressions before they spread.

Who it is for: Teams that need repeatable shader variant visibility in the editor and CI.

Product Docs License Store

Unity Serialization Migration Guard

Audit renamed fields, run scoped serialization coverage checks, preview reserialize results, and validate cleanup safety before removing legacy mappings.

Who it is for: Teams migrating serialized fields during refactors and large data model changes.

Product Docs License Store

IL2CPP Strip Guard

Scan for managed stripping, reflection, serialized type reference, and AOT generic risk patterns before IL2CPP build-only bugs appear.

Who it is for: Unity developers shipping IL2CPP targets such as WebGL, Android, iOS, VR, or console-like platforms.

Product Docs License Support

Inspector Event Link Doctor

Scan scenes, prefabs, ScriptableObjects, and animation clips for broken or risky Inspector-connected callbacks before runtime.

Who it is for: UI-heavy Unity teams, prefab-heavy projects, and developers validating callbacks before builds or refactors.

Product Docs License Support

Fast Play Mode Safety Guard

Scan Unity projects for static state, reset coverage, and runtime persistence risks before disabling Domain Reload.

Who it is for: Teams that want faster Play Mode entry without hidden static-state leaks.

Product Docs License Support

Import Settings Validator & Fix

Apply deterministic importer rule standards across texture, model, and audio assets, then validate and fix drift.

Who it is for: Teams standardizing importer behavior across contributors and machines.

Product Docs License Store

Third-Party Notices & Credits

Generate deterministic third-party notice, credits, and compliance manifest files for Unity release workflows.

Who it is for: Teams that need repeatable attribution outputs before shipping.

Product Docs License Store
