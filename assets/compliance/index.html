The Problem

Unity projects accumulate:

  • UPM packages
  • Asset Store plugins
  • Embedded dependencies
  • Native libraries
  • Copied utilities

At release time, you ask:

What licenses do we need to include?

Most teams guess.
Guessing is liability.

🔍 Evidence-Based Detection

The scanner looks for:

  • LICENSE files
  • NOTICE files
  • COPYING files
  • Package manifests
  • Known license markers

If it finds proof → it reports it.
If not → it marks UNKNOWN.

No assumptions.

  • UPM packages
  • Asset Store imports
  • Plugins folders
